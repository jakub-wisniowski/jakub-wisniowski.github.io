Planets

This app visualises assets from the Star Wars API. Allows users to sort assets by attributes when clicking table's header. The webpage uses pagination and allows users to modify characters' data. User can also add a new character via a form with validation set for every field.

Angular
Bootstrap
HTML5
CSS3
Jasmine

I've learnt the basics of Bootstrap and had a chance to use pagination for the first time. I used reactive Angular forms to validate user's input more efficiently. Last but not least I've learnt to write unit tests for Angular applications using mock components and mock services.

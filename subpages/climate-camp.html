Climate Camp

Application was designed for the attendees of Climate Camp 2019. It has four basic functions:

  • register for the camp
  • see event's program
  • display program's details
  • check the necessary things to pack for the camp

Flutter
Dart
Firebase

This was my first project in Flutter, I have learnt how to create views, use shared preferences and open external links. I have created my very first Firebase project in order to store information about camp's agenda and the list of crucial things to take to the camp.

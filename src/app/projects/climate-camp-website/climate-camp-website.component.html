Climate Camp website

The Climate Camp website is designed to gather all important information about the collective and it's actions. It has been released in July 2020 and is currently being enhanced by nice-to-have features. We are also working on improving the performance. It has been built by a team of three - UX designer, DevOps specialist and me - a Frontend Developer.

The website replaced the old Wordpress version - it is now possible to add content via a simple markdown editor set up by the DevOps specialist. My task was to come up with a suitable frontend architechture that would allow scully parse data from the editor and display them in the menu.

Angular
Bootstrap

I implemented the internationalisation using transloco package. I've been trying to optimize the loading speed by using static site generator - scully.io. Angular Animations have been used in the project to animate route changes. Right now I'm working on images optimization to make sure they don't slow down the website too much.

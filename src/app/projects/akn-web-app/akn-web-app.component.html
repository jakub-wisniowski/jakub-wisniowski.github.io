AKN web app

The AKN web app allows admin to create groups and assign teachers and students to them. Admins can create new accounts and change users' credentials. They can also remove users from groups. Teachers can create new lesson, assign it to a group and set exact time and date.

In the lesson view, teacher can add new words, edit them and remove them. Words can be added manually or automatically upon providing the app with a csv file. Teacher can also see student's progress in two modes: from last 3 lessons or from a specific lesson.
